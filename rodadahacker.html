RodAda Hacker

Skills: Digital security training, Graphic Design, Code training, Organization
URL: RodAdaHacker

RodAda Hacker is one of the first code events in Brazil only for women. In 2014 (Rio de Janeiro) and 2015 (São Paulo) I was coach. In 2015 in collaboration with Olabi and Sil Bahia I also collaborated as a producer, graphic designer and web developer. The event in São Paulo happened in 8 different places at the same time. For the first time all the trainers were women. This was one of our goals. :) During the event we had classes for many programming languages as: Ruby on Rails, Python, Arduino, PHP and Wordpress development.

I created and developed the website and an online toolkit.
